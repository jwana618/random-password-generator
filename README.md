# Password Generator (Python)

This is a simple Python password generator that creates a random password based on user preferences.

The user can choose:
- Password length
- Whether to include uppercase letters
- Lowercase letters
- Numbers
- Special characters

This project was built to practice Python basics and applying logic in a small project.

---

## Features

- Custom password length
- Optional character types:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Prevents generating a password if no character type is selected

---

## Technologies Used

- Python 3
- Built-in `random` module

---

## How to Run

1. Make sure Python is installed
2. Open the project folder
3. Run the file using:

```bash
python password_generator.py
```


4. Follow the prompts in the terminal
